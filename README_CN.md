## 何为 Swift X？

Swift 是一门与时俱进又不太稳定的语言，每次更新都有新的功能以及一些改动。

现在，Swift 的版本为 3.0.1（Xcode 8.2.1），以及 3.1（Xcode 8.3 beta）。今年晚些时候，4.0 也将登台。

所以我尝试使用 X 来概括 Swift 的版本，并及时更新代码以适配当前最新的 Xcode。
